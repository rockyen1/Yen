技術原理的不同
    3LCD 投影技術的原理是先利用投影機內的分色鏡將燈泡打出的白色光分解成R( 紅 )、G( 綠 )、B( 藍 ) 三種顏色 ( 光的三原色 ) 的光,接著三色光束需要分別透過 R、G、B 三色液晶板。在這個過程中,控制單元會通過控制液晶面板電路的開和閉合來 形成投影畫面所需的顏色、形狀和動作,最後這些「顏色、動作、形狀」經由稜鏡匯聚後 ,最終產生了我們在投影幕上看到的圖像。

    1-Chip DLP 投影技術
    DLP 投影的工作原理是將燈泡發射出的強光,經過均勻化處理後,讓其通過一個色輪(ColorWheel),色輪會將光分成 RGB 三色 ( 或者 RGBW 等更多色 ),而分色後的光線再由 透鏡投射在 DMD 芯片上,最後由 DMD 芯片來完成投影畫面動作、色彩的組合,經反射後 在投影螢幕上成像。
